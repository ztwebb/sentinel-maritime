## Project Overview

Sentinel Maritime is a 3D maritime situational awareness console designed to simulate the type of operational intelligence interfaces used in naval and maritime autonomy systems.

The platform visualizes vessel traffic, chokepoints, infrastructure, and behavioral signals to help operators understand maritime activity at both global and regional scales.

---

## Technology Stack

- Next.js
- TypeScript
- Three.js / React Three Fiber
- Zustand
- TailwindCSS

---

## Project Phases

### Phase 1 — Globe Foundation
- 3D Earth rendering
- vessel markers
- ports
- chokepoints
- intelligence panel
- theater navigation

### Phase 2 — Movement Intelligence
- vessel track trails
- AIS silence detection
- chokepoint stress index

### Phase 3 — Threat Context
- conflict events
- submarine cable infrastructure
- infrastructure proximity alerts

### Phase 4 — Analysis Tools
- timeline replay
- EEZ boundaries
